数据库第二天复习，直接整理题目来看，下面是复习内容。

一、数据库入门

选择题

1.下列选项中，退出MySQL命令行客户端的命令有（ ）

  • A、exit √
  • B、stop
  • C、drop
  • D、remove

2.下列选项中，约束的完整性要求主键不能重复，且不能取空值的是（ ）

  • A、域完整性
  • B、实体完整性 √
  • C、参照完整性
  • D、用户自定义完整性

3.下列选项中，用于停止MySQL服务的命令是（ ）

  • A、net stop
  • B、net start mysql
  • C、net stop mysql √
  • D、stop mysql

答案说明 :
net stop是MySQL服务的停止命令，mysql是需要停止的服务的名称。

4.下列选项中，用于初始化MySQL数据库的命令是（ ）

  • A、mysqld –initialize –console √
  • B、–initialize –console
  • C、mysql –initialize –console
  • D、mysql –initialize

答案说明 :
初始化MySQL数据库的命令是mysqld –initialize –console。其中，–initialize表示初始化数据库，–console表示将初始化的过程在控制台窗口中显示。

5、下列选项中，不属于DDL语句的是（ ）

  • A、CREATE语句
  • B、ALTER语句
  • C、DROP语句
  • D、SELECT语句 √

答案说明 :
数据库定义语言主要用于定义数据库、表等数据库对象，其中包括CREATE语句、ALTER语句和DROP语句。SELECT语句是数据查询语言。

6、下列选项中，用于安装MySQL服务的命令是（ ）

  • A、mysqld –install MySQL服务名称
  • B、mysqld MySQL服务名称
  • C、mysqld -install MySQL服务名称 √
  • D、mysqld -install

7、下列选项中，用于启动MySQL服务的命令是（ ）

  • A、net start
  • B、net start mysql √
  • C、net stop mysql
  • D、start mysql

答案说明 :
net start是MySQL服务的启动命令，mysql是需要启动的服务的名称。

8、下列选项中，对关系模型的完整性约束的描述错误的是（ ）

  • A、实体完整性要求关系中的主键不能重复且不能取空值
  • B、参照完整性定义了外键与主键之间的引用规则
  • C、实体完整性允许同一个数据表存在两条完全相同的记录 √
  • D、用户自定义完整性是用户针对具体的应用环境定义的完整性约束条件

答案说明 :
实体完整性要求同一个数据表中，不能存在两条完全相同无法区分的记录。

9、下列选项中，MySQL提供的默认用户是（ ）

  • A、admin
  • B、root √
  • C、user
  • D、test

10、下列选项中，属于非关系数据库中的键值存储数据库的是（ ）

  • A、Redis √
    B、HBase
    C、Neo4J
    D、MongoDB

答案说明 :
Redis是一个使用C语言编写的键值数据库。

11、下列选项中，用于卸载MySQL服务的命令是（ ）

  • A、mysqld –remove MySQL服务名称
  • B、mysqld MySQL服务名称
  • C、mysql –remove MySQL服务名称
  • D、mysqld -remove MySQL服务名称 √

12、下列选项中，对E-R图通用的表示方式描述错误的是（ ）

  • A、实体用矩形表示
  • B、属性用正方形框表示 √
  • C、联系用菱形框表示
  • D、用连线将相关的实体连接

答案说明 :
属性用椭圆框表示，将属性名写在椭圆框内。

13、下列选项中，不是数据库管理系统管理数据的层次是（ ）

  • A、外部层
  • B、中间层 √
    C、内部层
    D、概念层

14、下列选项中，不属于按照应用层次划分数据模型的是（ ）

  • A、概念数据模型
  • B、逻辑数据模型
  • C、物理数据模型
  • D、关系数据模型 √

判断题

15、数据库中的数据只包括普通意义上的数字和文字。

  • 正确
  • 错误 √

答案说明 :
数据库中的数据不仅包括普通意义上的数字，还包括文字、图像、声音等，也就是说，凡是在计算机中用来描述事物的记录都可称作数据。

16、E-R图是一种用图形表示的实体联系模型。

  • 正确 √
  • 错误

17、逻辑数据模型按照数据结构分为层次模型、线状模型、关系模型和面向对象模型。

  • 正确
  • 错误 √

答案说明 :
逻辑数据模型按照数据结构主要分为层次模型、网状模型、关系模型和面向对象模型。

18、MySQL官网提供了MySQL的企业版、高级集群版和社区版的产品供用户下载使用。

  • 正确 √
  • 错误

19、关系代数是一种抽象的查询语言，是研究关系模型的数学工具。

  • 正确 √
  • 错误

20、MySQL服务默认占用3306端口。

  • 正确 √
  • 错误

答案说明 :
MySQL服务默认监听3306端口，如果该端口被其他服务占用，会导致客户端无法连接MySQL服务器。

21、每个关系的数据结构是一张规范化的二维表，即关系模型通过二维表组织数据。

  • 正确 √
  • 错误

22、登录MySQL时，为了防止登录密码泄露，可以使用-p选项隐藏登录密码。

  • 正确 √
  • 错误

23、关系模型所约束的完整性通常包括实体完整性、参照完整性和用户自定义完整性。

  • 正确 √
  • 错误

24、SQLyog是一个高效、简洁的图形化管理工具，方便管理MySQL数据库。

  • 正确 √
  • 错误

填空题

25、退出MySQL命令行客户端使用 exit 或者 quit 命令。

答案说明 :
使用MySQL客户端成功登录MySQL后，如果需要退出MySQL命令行客户端，可以使用exit或者quit命令执行退出操作。

27、数据库管理系统用来管理数据的三个层次分别是 外部层 、概念层和内部层。

28、数据模型描述的内容有3部分，分别是数据结构、数据操作和 数据约束 。

29、数据管理技术发展的3个阶段分别是人工管理阶段、文件系统阶段和 数据库系统 阶段。

30、 E-R图 是一种用来描述现实世界的概念模型。

31、启动MySQL服务的命令是“net

start mysql MySQL服务名称”。

32、关系模型的完整性约束包括 实体完整性 、参照完整性和用户自定义完整性。

33、 SQL 是关系型数据库语言的标准，也是通用的关系数据库语言。

35、数据模型按照不同的应用层次划分，分为 概念数据模型 、 逻辑数据模型 、物理数据模型。

答案说明 :
数据模型按照不同的应用层次，主要分为概念数据模型（Conceptual Data Model）、逻辑数据模型（Logical Data Model）、物理数据模型（Physical Data Model）。

36、E-R图提供了表示实体、 属性 和联系的方法，用来描述现实世界的概念模型。

37、使用 help 命令可以查看客户端相关的帮助信息。

38、概念数据模型中实体与实体之间的联系，有一对一、 一对多 、多对多三种情况。
