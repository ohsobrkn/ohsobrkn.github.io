二、MySQL 基本操作

选择题

1、下列选项中，删除数据的关键字是（ ）

  • A、ALTER
  • B、DELETE √
  • C、DROP
  • D、UPDATE

2、下列选项中，不是MySQL自动创建的数据库是（ ）

  • A、information_schema
  • B、mysql
  • C、performance_schema
  • D、test √

3、下列选项中，删除数据表的语句是（ ）

  • A、DROP TABLE √
  • B、DELETE TABLE
  • C、REMOVE TABLE
  • D、ALTER TABLE

答案说明 :
在MySQL中，可以使用DROP TABLE语句删除数据表。

4、下列选项中，查看数据表创建信息的语句是（ ）

  • A、SHOW TABLES
  • B、DESC 数据表名
  • C、SHOW TABLE
  • D、SHOW CREATE TABLE 数据表名 √

答案说明 :
在MySQL中，可以通过SHOW CREATE TABLE语句显示创建数据表的语句。

5、下列选项中，查看指定数据库的创建信息的语句是（ ）

  • A、SELECR DATABASE TEST;
  • B、SHOW test;
  • C、SHOW CREATE DATABASE test; √
  • D、SELECT test;

6、下列选项中，语句“DELETE FROM student where name=’itcast’;”的执行结果是（ ）

  • A、只能删除name=’itcast’的一条记录
  • B、删除name=’itcast’的全部记录 √
  • C、只能删除name=’itcast’的最后一条记录
  • D、以上说法都不对

答案说明 :
WHERE子句用于指定删除的条件，满足条件的数据都会被删除。

7、下列选项中，关于UPDATE语句的描述正确的是（ ）

  • A、UPDATE只能更新表中的部分记录
  • B、UPDATE只能更新表中的全部记录
  • C、UPDATE语句更新数据时可以有条件的更新记录 √
  • D、UPDATE语句用于向数据表添加数据

8、下列选项中,用于查询数据的关键字是（ ）

  • A、INSERT
  • B、SELECT √
  • C、UPDATE
  • D、DELETE

9、下列选项中，对查询数据的描述正确的是（ ）

  • A、查询数据的SQL语句中多个字段名使用分号分隔 （逗号）
  • B、查询数据的SQL语句必须包含数据表的所有字段
  • C、通配符“*”表示查询数据表的所有字段名 √
  • D、通配符“#”表示查询数据表的所有字段名

10、下列选项中，能够实现修改数据表时，将字段id添加在数据表的第一列的语句是（ ）

  • A、ALTER TABLE dept ADD FIRST id INT;
  • B、ALTER TABLE dept ADD id INT FIRST; √
  • C、ALTER TABLE dept ADD AFTER id INT;
  • D、ALTER TABLE dept ADD id INT AFTER;

11、下列选项中，更新数据的SQL语句正确的是（ ）

  • A、UPDATE user SET id = u001;
  • B、UPDATE user(id,username) VALUES(‘u001’,’jack’);
  • C、UPDATE user SET id=’u001’,username=’jack’; √
  • D、UPDATE INTO user SET id = ‘u001’, username=’jack’

答案说明 :
MySQL中使用UPDATE语句更新表中的数据的语法： UPDATE 数据表名 SET 字段名1 = 值1[，字段名2 = 值2，…] [WHERE 条件表达式];

12、下列选项中，删除数据的SQL语句正确的是（ ）

A、DELETE student ,where id=11;

  • B、DELETE FROM student where id=11; √
    C、DELETE INTO student where id=11;
    D、DELETE student where id=11;

13、下列选项中，对于SQL语句“UPDATE student set name=’youjun’, grade=98.5；”的执行结果描述正确的是（ ）

  • A、更新student表中第一条记录
  • B、出错语法错误
  • C、更新student表中最后一条记录
  • D、更新student表中每一条记录 √

答案说明 :
在UPDATE语句中如果没有使用WHERE子句，则会将表中所有数据的指定字段都进行更新。

14、下列选项中，“SHOW TABLES LIKE ‘sh_’”语句的执行结果为（ ）

  • A、fish
  • B、mydb
  • C、she √
  • D、unshift

答案说明 :
LIKE子句，可以根据指定匹配模式匹配数据表，_”表示匹配一个字符，由此可知匹配结果为she。

15、下列选项中，用于向表中添加多条数据的关键字是（ ）

  • A、ALTER
  • B、CREATE
  • C、UPDATE
  • D、INSERT √

答案说明 :
在MySQL中可以使用INSERT语句向数据表中插入多条数据。

16、下列选项中，用于更新表中数据的关键字是（ ）

  • A、ALTER
  • B、CREATE
  • C、UPDATE √
  • D、REPLACE

答案说明 :
MySQL中使用UPDATE语句来更新表中的记录。

17、下列选项中，插入数据的语句错误的是（ ）

  • A、INSERT 数据表名 VALUE(值列表);
  • B、INSERT INTO 数据表名 VALUES(值列表);
  • C、INSERT 数据表名 VALUES(值列表);
  • D、INSERT 数据表名 (值列表); √

答案说明 :
向数据表中插入单条数据的语法格式如下所示。 INSERT [INTO] 数据表名 [(字段名1[, 字段名2] …)] {VALUES | VALUE} (值1) [, (值2)] …;

18、下列选项中，存储数据库和数据表的结构信息的数据库是（ ）

  • A、information_schema √
  • B、mysql
  • C、performance_schema
  • D、sys

19、下列选项中，删除user表中所有数据的SQL语句是（ ）

  • A、UPDATE user;
  • B、DROP TABLE user;
  • C、DELETE FROM user; √
  • D、DELETE user;

20、使用UPDATE语句更新student表id=1的记录，name更新值为”youjun”,grade更新值为98.5， 下列选项中，能够完成上述功能的SQL语句是（ ）

  • A、UPDATE student set name=youjun,grade=’98.5’ where id=1;
  • B、UPDATE student set name=’youjun’,grade=98.5 where id=1; √
  • C、UPDATE FORM student set name=’youjun’,grade=98.5 where id=1;
  • D、UPDATE student Values name=’youjun’,grade=98.5 where id=1;

答案说明 :
MySQL中使用UPDATE语句更新表中的数据的语法： UPDATE 数据表名 SET 字段名1 = 值1[，字段名2 = 值2，…] [WHERE 条件表达式];

  • 错误

  • 正确
  • 错误 √ （应为逗号）

24、ALTER TABLE语句中的CHANGE子句仅可以修改字段名。

  • 正确

  • 错误 √ （可以修改字段类型、约束、默认值等）

    答案说明 :
    CHANGE子句不仅可以修改字段名称，还可以重新定义字段的数据类型、约束、排列位置。

25、

使用INSERT语句插入数据时省略所有字段名，添加的值的顺序必须和字段在表中定义的顺序相同。

  • 正确 √

  • 错误

    答案说明 :
    使用INSERT语句插入数据时，如果没有指定字段名，添加的值的顺序必须和字段在表中定义的顺序相同。

  • 正确 √
  • 错误

27、MySQL提供DELETE语句用于删除表中的数据。

  • 正确 √
  • 错误

28、使用INSERT语句插入数据时，字段名是可以省略的。

  • 正确 √
  • 错误

29、创建的临时表仅在当前会话可见，会话关闭时会自动删除。

  • 正确 √
  • 错误

30、在MySQL中，通过DESC语句可以查看数据表的结构信息。

  • 错误
