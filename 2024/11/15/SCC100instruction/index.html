SCC100客制化键盘说明书

写在开头

第二把客制化键盘，火星之钥SCC100闪粉奶白🥰性价比真蛮高的，清风福利团购买，团长代组体验拉满了🥰

键盘设置

  1. 长按3秒Fn+Backspace，键盘恢复出厂设置，整体背光红-绿-蓝，闪烁3次。
  2. Fn +M 切换Win/MAC系统
    • 切换到Win系统CAPSLOCK键蓝光闪烁3下 指示，切换到Mac系统白光闪烁3下指示。
  3. Fn+win 锁住win键
  4. 驱动设置5档按键延迟,默认第三档，按需调整。
  5. 在未连接USB的情况下，直接用FN+Z/X/C/V切换2.4G/BT1/BT2/BT3。
    • FN + Z—短按切换到蓝牙设备一，长按3S进入配对。
    • FN + X—短按切换到蓝牙设备二，长按3S进入配对。
    • FN + C—短按切换到蓝牙设备三，长按3S进入配对。
    • FN + V—长按 3S 入2.4G配对接收器模式。
  6. FN+Delete—查询当前电量
    • “0~1”10颗LED显示10档电量，三档及以下亮红色，三档以上亮绿色。

FN 组合键 灯光调整模式

灯效:

  1. 关灯
  2. 常亮模式
  3. 呼吸模式
  4. 梦幻彩虹
  5. 一触即发
  6. 雨中散步
  7. 彩虹轮盘
  8. 按键涟漪
  9. 繁星点点
  10. 川流不息
  11. 随波逐流（默认）
  12. 如影随形
  13. 正炫光波
  14. 旋转风车
  15. 七彩瀑布
  16. 花开富贵
  17. 游戏灯光：默认FPS模式: ESC.W/A/S/D/↑/↓/←/→，亮蓝灯，使用驱动可定义。

灯光调整快捷键：

  1. FN + Q 关闭/打开按键灯光
  2. FN + W 上一个灯光效果
  3. FN + E 下一个灯光效果
  4. FN + R 灯光速度加快
  5. FN + T 灯光速度减慢
  6. FN + A 灯光亮度增加
  7. FN + S 灯光亮度减少
  8. FN+D 背光颜色切换：混彩-红-绿-蓝-黄-紫-青-白
  9. FN + F 轴灯/灯条切换 按第一下关闭轴灯仅打开灯条，按第二下关闭灯条打开轴灯，按第三下轴灯和灯条一起亮
  10. FN + 右ALT = 切换Logo灯效：流光-霓虹-常亮(默认红色)-呼吸(默认红色)-关闭。
  11. FN + /? = 切换Logo灯颜色：红-绿-蓝-黄-紫-青-白,常亮&呼吸模式下有效。
